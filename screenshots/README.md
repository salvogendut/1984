# Screenshots

One screenshot per supported platform, named after the platform:

| File | Shown in README as |
|---|---|
| `linux.png` | Linux |
| `windows.png` | Windows |
| `haiku.png` | Haiku |
| `netbsd.png` | NetBSD |
| `openbsd.png` | OpenBSD |
| `macos.png` | macOS |

Capture the emulator window with a recognisable program running (a game with on-screen text, the BASIC prompt with some output, or the F9 overlay) so the platform's native window chrome is visible alongside the CPC display. PNG, no specific size requirement — the README renders them as 280-px thumbnails.
